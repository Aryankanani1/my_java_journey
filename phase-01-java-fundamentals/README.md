# Phase 1 — Java Fundamentals

The foundation. Everything else in this journey depends on getting this solid.

## Goals

By the end of this phase I should be able to:
- Write a small Java program using OOP, collections, streams, and exception handling
- Read and understand any basic Java codebase
- Write unit tests with JUnit 5
- Use Git and IntelliJ confidently

## Topics

1. **Setup & tooling** — JDK, IntelliJ, Git, GitHub
2. **Core syntax** — types, operators, control flow, arrays, strings, methods
3. **OOP** — classes, inheritance, polymorphism, interfaces, records
4. **Exception handling** — checked vs unchecked, try-with-resources
5. **Collections** — List, Set, Map, Queue
6. **Generics** — bounded types, wildcards, PECS
7. **Modern Java (8+)** — lambdas, streams, Optional, records
8. **File I/O** — NIO.2, buffered I/O
9. **Concurrency basics** — threads, executors, CompletableFuture
10. **Testing** — JUnit 5, assertions, test naming

## Notes

I'll keep a running `notes.md` in each topic folder with things I want to remember
or concepts that confused me the first time.
