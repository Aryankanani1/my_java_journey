# Setup & First Commit

## Checklist

- [ ] JDK 21 installed (`java -version` shows 21.x.x)
- [ ] IntelliJ IDEA Community installed
- [ ] Git installed (`git --version`)
- [ ] GitHub account created
- [ ] SSH key added to GitHub (or HTTPS with token)
- [ ] Git configured with name and email
- [ ] This repo cloned and first commit pushed

## Verifying Java works

```bash
cd phase-01-java-fundamentals/01-setup
javac HelloWorld.java
java HelloWorld
```

Expected output: `Hello, Java journey!`

## Git config (one-time)

```bash
git config --global user.name "Your Name"
git config --global user.email "you@example.com"
```
